# 🏆 IPL Analysis Dashboard (2008-2025)

---

## 📝 Introduction
The **IPL Analysis Dashboard** is a comprehensive data analytics project focused on exploring and visualizing key insights from the **Indian Premier League (IPL)** dataset. This project was developed to analyze team performances, player statistics, match trends, venue-wise outcomes, and seasonal patterns using interactive business intelligence tools.

The goal of this project is to transform raw IPL data into meaningful, actionable insights through engaging dashboards and visual storytelling.

## 🎯 Project Objective
- Analyze overall IPL match trends across multiple seasons
- Evaluate team-wise and player-wise performance
- Identify winning patterns based on toss, venue, and batting order
- Compare seasonal statistics and consistency across teams
- Build an interactive dashboard for easy exploration of cricket data

## 🛠️ Tools & Technologies Used
- **Power BI** – Data visualization & dashboard development  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Advanced calculations and measures  
- **Excel / CSV Dataset** – Data source handling  
- Data Modeling

## 📂 Dataset Description
- **ball_by_ball_data.csv** - This dataset contains every data related to per ball bowled in all the seasons from 2008-2025. This contains bowler information who was bowling, batter information who was batting, Non-striker end batsman name, Over number, total runs, Runs information etc.

- **ipl_matches_data.csv** - This dataset contains matches data like which team won which match, match venue, match date, toss winner, toss decision, win_by_wicket, win_by_run etc.

- **teams_data** - This one contains team_id, team_name. this is a small dataset, which is helipng us to get the team_id, name and team short name and the image url used in the dashboard designing.

- **players_data_updated** - This dataset contains every player_id, player_name, bat_style, bowl_style.

## 📊 Dashboard Features

- **Player Insights**
  - Top run Scorer(Orange Cap holder)
  - Leading wicket taker(Purple Cap holder)
  - Most sixes scored by the player
  - Most Fours scored by the player

- **Team Insights**
  - Matches Played
  - Wins & losses
  - Total points scored

- **Seasonal Analysis**
  - Champions Team
  - Runner Up Team
  - Total Sixes & fours scored in particular season
  - Total Matches played in the season
  - Total teaams participated
  - Centuries & half-centuries scored in a season
  - Total venues

## 📈 **Key Insights**
- Certain teams demonstrate strong consistency across seasons.
- Toss decisions can significantly influence match outcomes.
- Venue conditions often impact batting and bowling performances.
- A small group of players dominate scoring and wicket-taking charts.
- Seasonal analysis highlights shifts in team strategies over time.
